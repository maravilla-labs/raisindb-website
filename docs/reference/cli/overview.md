# CLI Overview

RaisinDB command-line interface.

## Installation

```bash
npm install -g raisindb-cli
```

## Commands

- [package](./commands.md#package-create) - Package management
- [clone](./commands.md#clone) - Download packages
- [sync](./commands.md#sync) - Bidirectional sync

## Global Options

- `--server <url>` - Server URL
- `--repo <name>` - Repository name
- `--help` - Show help
- `--version` - Show version
